# socket.io-chat-app
A simple chat application using socket.io
## Usage

### Clone
Clone the Project

```sh
$ git clone https://github.com/tanishha/socket.io-chat-app.git
$ cd socket.io-chat-app
```

### Installation

Install the dependencies

```sh
$ npm install
```

### Run

This will run your dev server at http://localhost:3000

```sh
$ npm start
```
